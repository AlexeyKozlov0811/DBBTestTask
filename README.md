DBBTestTask
====

### Local Install

Get code from repository:
https://github.com/AlexeyKozlov0811/DBBTestTask

We use `pipenv`
https://pipenv.pypa.io/en/latest/ (to install it on Ubuntu/Debian:
`sudo apt install pipenv`)

Install Python packages

```bash
pipenv install -d
```

### Run Local Server
```bash
python main.py
```
